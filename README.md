## Biketrip Analysis

A data analysis project for rides for the past 12 months in 2023, of members and casual rides done with MySQL and R with some graphics from Tableau.

The data was sourced from [here](https://divvy-tripdata.s3.amazonaws.com/index.html), The data has been made available by
Motivate International Inc. under this [license](https://www.divvybikes.com/data-license-agreement)
